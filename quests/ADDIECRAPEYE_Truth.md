# Truth Elearning Module

## Overview
A RUST-based, Vite+React elearning module that gently "pops the bubble" of truth on understanding limitations. Designed to challenge learners' perceptions with a mix of cheeky humor and deep introspection.

## Core Objectives
- **Design**: Create a React/Vite frontend with interactive elements
- **Reflection**: Force users to confront their own cognitive biases
- **Engagement**: Use RUST backend for performance + cheeky AI personality
- **Delivery**: Deploy as a standalone web module with embedded WASM
