## First Quest: "Clarify a Problem" (Needs Assessment)

**Quest Description:**
Before building anything, an expert instructional designer diagnoses the root problem. Is this a training issue? Or is it a workflow problem? A motivation problem? A tool problem?

**Objectives (5 steps):**
1. **Identify the Gap** — What business/learning outcome is not being achieved?
2. **Describe the Audience** — Who are the learners? Novices? Experts? Mixed?
3. **Analyze the Environment** — Where will learning happen? What constraints exist?
4. **Determine If Training Is the Answer** — Would a job aid work better? A process change?
5. **Frame the Problem** — Articulate the specific, measurable problem to solve

**Rewards:**
- +50 XP toward 🔍 Perception (Planning & Analysis domain)
- Unlock skill: **Diagnostic Vision** (ATD #2)
- Unlock P-ART-Y role: Researcher (for testing sandbox)

**SME Interview Questions (STAR Method):**
- What business need does this address?
- What challenges do learners face?
- Can you describe a specific incident (Situation, Task, Action, Result)?
- How would you explain this to an 8-year-old?

## XP & Leveling Logic

**XP Sources:**
- +25 XP per quest objective completed
- +100 XP per phase completed (Analysis → Design → etc.)
- +50 XP per skill unlocked (ATD abilities)
- +200 XP per domain level up (IBSTPI)
- +500 XP per project completed (full ADDIECRAPEYE cycle)

**Domain Level Thresholds:**
- Level 1 (Novice): 0-500 XP
- Level 2 (Advanced Beginner): 500-1500 XP
- Level 3 (Competent): 1500-3000 XP
- Level 4 (Proficient): 3000-6000 XP
- Level 5 (Expert): 6000+ XP

**Resonance Level (Meta-Progression):**
- Each completed project adds +1 Resonance
- Resonance unlocks advanced P-ART-Y capabilities
- Resonance 5 = "Cognitive Architect" title
- Resonance 10 = "Scope Recycler Supreme" (max)
