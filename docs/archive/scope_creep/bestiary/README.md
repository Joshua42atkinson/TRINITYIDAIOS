# The Dumpster Universe Bestiary

Welcome to the **Dumpster Universe Bestiary**. This is not a trash bin—it is a library of inspiration and a graveyard of ambitious ideas waiting to be resurrected.

In the LitRPG narrative, these are the "Scope Creep" monsters. They are manifestations of raw, unbridled creativity that were deemed too dangerous or resource-heavy to run in the current main build, but are too valuable to lose. 

By banishing them to the Bestiary, we honor the creator's vision and preserve their architectural detours as "wild beasts" that a seasoned player can later "Solo Level" and tame.

## The Bestiary Template

Use this template to catalog new Scope Creeps:

```markdown
# [Monster Name]
*A manifestation of [Specific Feature/Idea]*

**HP (Tech Debt):** [1-100] (How entangled this code is with the main system. Higher means harder to tame.)
**Mana Cost:** [Hardware/VRAM requirement]
**Taming Requirement:** [Which ADDIECRAPEYE phase needs to be passed to safely merge this]
**Git Artifacts:** [Path to archive or Git branch name]

### Lore (The Origin Story)
[Why was this created? What was the original daydream? Why was it banished to the Dumpster?]

### Integration Strategy (The Taming Plan)
[How would a high-level developer eventually safely merge this back into the Iron Road?]
```
