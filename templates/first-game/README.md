# First Game Template - 2D Educational Puzzle

This template is generated by the Trinity AI OS quest chain:
1. Artist sidecar → Game Design Document
2. Engineer sidecar → Bevy implementation
3. Evaluator sidecar → QM rubric compliance check

## Structure

```
templates/first-game/
├── design.md          # GDD from Artist
├── src/
│   ├── main.rs       # Bevy game entry point
│   ├── game.rs       # Game logic
│   ├── ui.rs         # UI components
│   └── assets.rs     # Asset loading
├── assets/           # Generated game assets
├── Cargo.toml        # Dependencies
└── evaluation.md     # QM rubric report
```

## Quest Chain

To generate this template, execute:

```bash
# 1. Start Artist sidecar
./target/release/trinity-sidecar-engineer --role artist &

# 2. Execute design quest
curl -X POST http://127.0.0.1:8090/quest/execute \
  -d '{"quest_id":"quest-design-first-game"}'

# 3. Wait for completion (5-10 min with timeout safety)
# 4. Review design.md output
# 5. Start Engineer sidecar and generate code
# 6. Start Evaluator sidecar and validate
```
